## rest-based backend for paideia.im
- FastAPI Python


## Quick Start
Clone the repo
```
$ cd paideia-api
$ docker compose up
```

- Make sure you have the appropriate dev environment setup
- Depends on ergonode, ergoexplorer, postgresql and redis


## Testing
Run Unit Tests
```
$ cd paideia-api/app
$ python3 -m pytest
```
